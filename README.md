# Webapp
cách cài đặt:
1. chạy npm install hoặc yarn install nếu bạn sử dụng yarn
2. chạy bower install để cài đặt bower

cách chạy:
##   chạy live view:
```javascript
gulp serve
```
#   render:
```javascript
gulp serve:dist
```

chương trình webapp của Yeoman, tích hợp gulp cùng cấu trúc folder cho Sass
